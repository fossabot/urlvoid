URLVoid API Gem

Ruby Gem for URLVoid API.

Installation

Add this line to your application's Gemfile:

gem 'urlvoid'

And then execute:

$ bundle

Or install it yourself as:

$ gem install urlvoid

Usage

URLVoid Host Response

require 'urlvoid'

api_key = 'MY_API_KEY'

urlvoid = URLVoid::Host.info(api_key, 'google.com')

URLVoid::Stats.remaining_queries(api_key)
# => 901
